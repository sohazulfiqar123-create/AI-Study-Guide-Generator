# 📚 AI Study Guide Generator

An AI-powered study assistant built with **n8n** and **OpenAI**. This workflow transforms study notes into a complete study guide within seconds.

## 🌐 Live Demo

🔗 **Try it here:**  
https://amnaali123.app.n8n.cloud/form/study-guide

## ✨ Features

- 📄 Generates a concise summary
- 🔑 Extracts key points
- ❓ Creates 10 MCQs with answers
- 📝 Generates short-answer questions
- 📚 Lists important concepts and definitions
- ⚠️ Highlights common exam mistakes
- ⏰ Creates a 30-minute revision plan

## 🛠️ Tech Stack

- n8n
- OpenAI API
- n8n Form Trigger

## 🚀 How It Works

1. Open the study guide form.
2. Enter the subject.
3. Paste your study notes.
4. Submit the form.
5. Receive a complete AI-generated study guide instantly.
